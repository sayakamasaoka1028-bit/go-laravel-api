# go-laravel-api

認証とデプロイメントを念頭に、Go と Laravel を使用したバックエンド API の実践。

## Purpose
This repository is for practicing a backend architecture using:
- Go for API services
- Laravel for application and management layer

## Tech Stack

- Laravel
- REST API
- Authentication
- Deployment
